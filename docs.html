NOVA Language Documentation

1. Overview

NOVA is a simple, readable, custom programming language designed for clarity and ease of use.

2. Variables

Variables in NOVA are created and assigned in two steps.

2.1. define

Creates a variable with no value (null).

define hp
define name
define score

2.2. set

Assigns a value to an already defined variable.

set hp = 30
set name = "Xylen"
set score = 100

2.3. Text values

Text must always be wrapped in quotation marks.

set name = "Xylen"
set sound = "meow"

If quotes are missing, NOVA treats the text as a variable name and throws an error.

set name = meow
[ERROR] meow is not defined

2.4. Showing values

To display a variable, use show value.

define hp
set hp = 30

show value hp

3. Output

3.1. Text output

show "Hello"

3.2. Variable output

show value hp

4. Conditions

define hp
set hp = 30

when hp > 10
    show "OK"
otherwise
    show "LOW"
end

5. Loops

5.1. Repeat

repeat 3 times
    show "attack!"
end

5.2. Iterate

over x in ["A","B","C"]
    show value x
end

6. Functions

do heal(target, amount) -> number
    return target.hp + amount
end

7. Structures

shape Player
    name: text
    hp: number
    alive: flag = yes
end

8. Example Program

define hp
define name

set hp = 30
set name = "Xylen"

show value hp
show value name

repeat 3 times
    show "attack!"
end

9. Summary: define, set, show value

define hp        ; creates variable
set hp = 30      ; assigns value
show value hp    ; prints value

pkg nova.js --targets node18-win-x64 --output nova.exe

nova.exe program.nova
